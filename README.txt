Hangman - Dennis Tijbosch & Shaun van Beurden

Om het programma uit te voeren op je Linux systeem, moet je de volgende stappen doorlopen:

1) Zet hangman.sh en de /lang directory in je bin map
2) Zip "hangman.6" -> gzip hangman.6
3) Copy de zip naar de juiste manpage map -> sudo cp hangman.6.gz /usr/share/man/man6

Ons programma zit in sectie 6 (Games).
